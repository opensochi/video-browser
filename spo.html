Исходный код

Исходный код этого сайта хостится на GitHub.
Ссылка: https://github.com/xtsidx/Transparent-YouTube-Chanel

Используются следующие компоненты

JQuery

JavaScript фреймворк для работы с DOM элементами на странице, анимацией, AJAX. В своём составе имеет и другие полезности для программиста.
Сайт проекта: http://jquery.com

Backbone.js

Легковесный JavaScript фреймворк позволяет с небольшими усилиями написать MVC проект с активным использованием технологии AJAX.
Сайт проекта: http://documentcloud.github.com/backbone

Leaflet

JavaScript библиотека для добавление интерактивных карт на сайт, отличается простотой и малым размером.
Сайт проекта: http://leaflet.cloudmade.com

SmartPaginator

Для постраничной навигации по контенту на сайте используется jQuery плагин SmartPaginator.
Сайт проекта: http://www.egrappler.com/jquery-pagination-plugin-smart-paginator

Использовались следующие онлайн сервисы

Preloaders.net

Онлайн сервис для генерации GIF и APNG анимированных изображений, для отображения загрузки. Много шаблонов и настроек.
Сайт: http://preloaders.net
